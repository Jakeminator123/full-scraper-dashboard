Vad betyder delarna?

Tre faser körs för att bygga en komplett persondatabas med robust resume vid redeploy.
  • Fas 0 (Ratsit Harvesting): söker per födelsedatum + kön, löser PNR och fyller på \"snabba träffar\".
  • Fas 1 (Fordon): går prefix AAA-ZZZ och samlar fordonsdata/ägarkopplingar från biluppgifter.
  • Fas 2 (PNR brute-force): systematisk Luhn-baserad körning som fyller luckorna till målantalet.
  • Klar-prognos: ETA räknas mot målantal personer (default 10,4M om inget target skickas).
  • Redeploy-säkerhet: checkpoint + auto-resume gör att jobb fortsätter efter omstart.
